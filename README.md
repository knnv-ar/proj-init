### Movement v1.0 variable font

#### Layout features: `aalt`, `calt`, `kern`, `mark`

- `aalt`: Access All Alternates [_off by default_]
- `calt`: Contextual Alternates [_on by default_]
- `kern`: Kerning [_on by default_]

#### Variable standard (lowercase) axes:

- `wght`: 100-900

#### Variable custom (uppercase) axes:

- `SPAC`: 100-120

#### Instances (6):

- `Direct Thin`
  ```css
  .movement-regular-direct-thin {
      font-variation-settings: "wght" 100, "SPAC" 100;
  }
  ```
- `Indirect Thin`
  ```css
  .movement-regular-indirect-thin {
      font-variation-settings: "wght" 100, "SPAC" 120;
  }
  ```
- `Direct Black`
    ```css
  .movement-regular-direct-black {
      font-variation-settings: "wght" 900, "SPAC" 100;
  }
  ```
- `Indirect Black`
    ```css
  .movement-regular-indirect-black {
      font-variation-settings: "wght" 900, "SPAC" 120;
  }
  ```
- `Direct Light`
  ```css
  .movement-regular-direct-light {
      font-variation-settings: "wght" 250, "SPAC" 100;
  }
  ```
- `Indirect Light`
  ```css
  .movement-regular-indirect-light {
      font-variation-settings: "wght" 250, "SPAC" 120;
  }

- `Custom instance`
  ```css
  .movement-regular-custom-instance {
      font-variation-settings: "wght" 200, "SPAC" 110;
  }
  ```

 * Generated by Wakamai Fondue - https://wakamaifondue.com
 * by Roel Nieskens/PixelAmbacht - https://pixelambacht.nl
 */

/* Set custom properties for each layout feature */
:root {
    --movement-regular-aalt: "aalt" off;
}

/* If class is applied, update custom property and
   apply modern font-variant-* when supported */
.movement-regular-aalt {
    --movement-regular-aalt: "aalt" on;
}

/* Apply current state of all custom properties
   whenever a class is being applied */
.movement-regular-aalt {
    font-feature-settings: var(--movement-regular-aalt);
}

/* Variable instances */
.movement-regular-direct-thin {
    font-variation-settings: "wght" 100, "SPAC" 100;
}

.movement-regular-indirect-thin {
    font-variation-settings: "wght" 100, "SPAC" 120;
}
